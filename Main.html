Description

The API is vaguely RESTful. Here's an overview:

GET /api/
get names of lists (TODO: consider changing this endpoint)
GET /api/:list
get all items in :list
POST /api/:list {item}
add {item} to :list
PUT /api/:list/:id {item}
replace item :id with {item} in :list
DELETE /api/:list/:id
remove item :id from :list

See the Todo module for implementation details.
